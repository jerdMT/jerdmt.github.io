Predicting Tree Mortality with XGBoost

Background on Data: Tree Mortality

In the past decade, forest fires have increased due to climate change and fire suppression policies, with bark beetle infestations playing a notable role. Prescribed burns are used to reduce fuel and lower the risk of more intense fires while improving overall forest health. Bark beetles can lead to increased fire risk and negatively impact forest health by killing trees and reducing carbon absorption. The US Forestry Serviceâ€™s FTM database compiles data on tree mortality from both prescribed and naturally occurring fires. A subset of this database includes tree descriptors, health status, and beetle species presence over a decade. This information can be used as input in supervised learning models to predict if a given tree will survive up to three years based on its current state.

Preprocessing

Many variables are missing across many tree records. Only features with 50% or more of all measurements recorded are used as input variables. Numerical features are imputed with the mean and categorical variables are encoded.

from xgboost import XGBClassifier

#Hyperparameter Tunining with Grid-Search
def get_best_XGB_model(X_train,y_train,params):
    clf = XGBClassifier(seed = seed)
    gcv = GridSearchCV(estimator=clf, param_grid=params, scoring = 'roc_auc')
    gcv.fit(X_train, y_train)
    model = gcv.best_estimator_
    return model

#note: sklearn.metrics.confusion_matrix uses class in sorted order so 0,1 --> Dead, Alive
def evaluate_classifier(clf, X_train, y_train, X_test, y_test):
    clf.fit(X_train, y_train)
    y_pred = clf.predict(X_test)
    confusion_mat = confusion_matrix(y_test, y_pred)
    accuracy = accuracy_score(y_test, y_pred)
    fpr, tpr, thresholds = roc_curve(y_test, y_pred)
    auc = roc_auc_score(y_test, y_pred)
    disp = ConfusionMatrixDisplay(confusion_mat, display_labels=['Dead','Alive'])
    return accuracy, disp, fpr, tpr, auc
